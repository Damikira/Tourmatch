# Tourmatch

En tu terminal de Visual Studio Code, asegúrate de estar dentro de la carpeta donde pegaste tu pom.xml. Si estabas en la raíz de Tourmatch, escribe esto para entrar a la carpeta del backend:
cd proyecto/backend
Una vez ahí, escribe el siguiente comando y presiona Enter:
mvn wrapper:wrapper
y por ultimo ejecutas el comando
mvn spring-boot:run
